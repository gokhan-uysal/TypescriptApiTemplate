# Typescript Express Server Build & Run Guide

<h3>Build For Development</h3>
<ul>
<li>npm install</li>
</ul>

<h3>Build For Cloud</h3>
<ol>
<li>npm run clear</li>
<li>npm run build</li>
</ol>

<h3>Run For Cloud</h3>
<ul>
<li>npm start</li>
</ul>

<h3>Run For Development</h3>
<ul>
<li>nodemon</li>
</ul>
